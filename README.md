# Marca Creatives Marketing Website

A premium, interactive marketing website for **Marca Creatives**—a full-stack content and strategic branding agency building B2B business marketing assets and high-end personal branding profiles.

## Project Features
- **GSAP & ScrollTrigger Animations**: Scrub-controlled rotation, stagger fade-reveals, and hardware-accelerated movement waves.
- **Dedicated About Page**: Featuring a circular spinning SVG text badge and grayscaled modern team cards.
- **Horizontal Flex Sliders & Vector Icons**: Clean, light-weight inline SVG outline icons scaling across devices.
- **Fully Responsive Designs**: High-fidelity overlays collapsing gracefully on mobile viewports.

## Tech Stack
- HTML5, CSS3 Custom Properties
- GSAP, ScrollTrigger CDN
- Vanilla JavaScript
- Python HTTP Server (local environment testing)
